LearnWebGPU - Code
==================

This repository contains the reference code base accompanying the [Learn WebGPU](https://eliemichel.github.io/LearnWebGPU/) web book.

Branch `step028`: This corresponds to the code at the end of the page [C++ wrapper](https://eliemichel.github.io/LearnWebGPU/getting-started/cpp-idioms.html).

Building
--------

```
cmake . -B build
cmake --build build
```

Then run either `./build/App` (linux/macOS/MinGW) or `build\Debug\App.exe` (MSVC).
